# behavior identical.

# requests). 2.32.3 patches it.

# in 22.0.0; bump to the latest stable 23.x.

# reportlab powers the Step 6 "Download PDF" button. Pure-Python wheel,

# openpyxl reads PCLaw Excel (.xlsx) exports so firms can upload reports
# straight from Excel without re-saving as CSV (see excel_convert.py). Pure
# Python, no system libraries — fine on Render's slim image.
openpyxl==3.1.5
# python-dotenv backs the `from dotenv import load_dotenv` call at the top of
# app.py. It is imported unconditionally at module import time, so a missing

# fails before serving any request. Pinned to the current stable 1.x.
python-dotenv==1.0.1
